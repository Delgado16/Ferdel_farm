¿Cómo funciona?

El sistema distribuye el abono en cascada (FIFO) de manera automática y transparente para ahorrarte tiempo:

  1. Busca todas las facturas pendientes de cobro del proveedor seleccionado ordenadas por fecha de vencimiento.
  2. Aplica el abono de forma secuencial, liquidando primero las facturas más antiguas.
  3. Por cada factura afectada, se guardará un desglose detallado con el saldo anterior, el monto aplicado y el saldo nuevo.
Asegúrate de que el monto a abonar no supere el saldo total adeudado.
